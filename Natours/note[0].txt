 WHAT DOES HAPPENS BEHIND THE SEEN OF CSS PARSE.

a. css terminology
      .selectors             .declaration block
      .declaration           .property and declared value

b. first step, resolve conflict css declarations(cascade)
     .is process of combining different stylesheets and resolving different conflicts between different
          CSS rules and declarations, when more than one rule applies to a certain elements.

      We have three main css declaration
        1. The authors declaration
        2. User declarations
        3. browser(user agent) just default css


    -------  IMPORTANCE ----------

      1. USER ! important declarations
      2. Author !important declarations
      3. Author declarations
      4. User declarations
      5. Default browser declarations

      example
           .button{
             font-size: 20px;
             color: white;
             background-color: blue !important;    this will preceed than the
                                                   lower declaration
           }

           #nav .pull-right .button{
               background-color: green;
           }


    ------- SPECIFICITY --------
        1. inline styles
        2. IDS
        3. Classes, Pseudo-classes, attribute
        4. Elements, pseudo-elements

        example:
            .button{
                font-size: 20px;
                color: white;
                background-color: blue;
            }

            specificity(0 , 0 ,1, 0)

            nav#nav div.pull-right .button{
                background-color: green;
            }
            specificity(0 , 1 ,2, 2)   this has more specificity
                                        so the link does take the background
                                        of this selector.this wining value is called the
                                        cascade value.

            a{
                background-color: purple;
            }
            specificity(0 , 0 ,0, 1)

            #nav a.button:hover{
            background-color: yellow;
            }
            specificity(0 , 1 ,2, 1)

  ----------SOURCE ORDER  -------------------

  if all the selectors has the same importance and  same specificity
    the last declaration in the code will override all the other declarations
    and will be applied.

##### general
IMPORTANCE > SPECIFICITY > SOURCE CODE
